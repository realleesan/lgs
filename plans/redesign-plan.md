# Kế hoạch thiết kế lại website Misty LGS

## 1. Phân tích vấn đề hiện tại

### 1.1 Nội dung
- Quá nhiều text, lặp lại ý nhiều lần
- Thiếu tính tinh gọn, đi thẳng vào vấn đề
- Copywriting chưa thuyết phục, thiếu CTA rõ ràng

### 1.2 Thiết kế
- Bố cục quá phổ thông, giống mọi website truyền thống
- Thiếu điểm nhấn, không thu hút
- Không có visual hierarchy rõ ràng

### 1.3 Cấu trúc trang
- Tất cả các trang đều có cấu trúc giống nhau: heading → text → grid → CTA
- Thiếu sự đa dạng về bố cục
- Không có sự khác biệt giữa các section

---

## 2. Nguyên tắc thiết kế mới

### 2.1 Phong cách SaaS hiện đại
- **Minimalism**: Ít text, nhiều visual
- **Typography-driven**: Sử dụng font làm điểm nhấn
- **Whitespace**: Khoảng trắng lớn tạo cảm giác cao cấp
- **Gradients**: Sử dụng màu gradient tinh tế
- **Micro-interactions**: Hiệu ứng nhỏ tạo trải nghiệm

### 2.2 Color Palette (giữ nguyên)
- Primary: #356df1 (xanh dương)
- Background: #ffffff (trắng)
- Text: #000000 (đen), #666666 (xám)

---

## 3. Cấu trúc trang mới

### 3.1 Trang chủ (Homepage)

**Section 1: Hero**
- Thay đổi: Bỏ grid 2 cột → Layout-centered hoặc layout asymmetrical
- Tiêu đề ngắn gọn, mạnh mẽ (thay vì dài dòng)
- CTA rõ ràng, nổi bật
- Background: Gradient hoặc abstract shape

**Section 2: Social Proof**
- Logo carousel các đối tác/khách hàng
- Testimonials ngắn gọn
- Trust badges

**Section 3: Features (thay thế Services)**
- 3-column nhưng với layout khác: Icon → Title → Short desc
- Hover effects tinh tế
- Không cần nhiều text

**Section 4: Stats/Impact**
- Big numbers với animation
- Ít nhưng ấn tượng

**Section 5: CTA Section**
- Khác biệt hoàn toàn: Dark background, contrasting text

### 3.2 Trang Dịch vụ (Services)
- Thay đổi: Bố cục alternating (trái-phải, phải-trái)
- Mỗi dịch vụ có cấu trúc khác nhau
- Sử dụng visual lớn hơn

### 3.3 Trang Giải pháp (Solutions)
- Thay đổi: Tab-based layout hoặc accordion
- Tập trung vào use cases
- Ngắn gọn, dễ scan

### 3.4 Trang Báo giá (Pricing)
- Giữ nguyên nhưng tinh gọn hơn
- So sánh side-by-side
- Less text, more visual

### 3.5 Trang Tin tức (News)
- Grid với size khác nhau (masonry-style)
- Hover effects đặc biệt

### 3.6 Trang Liên hệ (Contact)
- Split layout với map lớn hơn
- Form đơn giản hóa

---

## 4. Các điều chỉnh cụ thể

### 4.1 Typography
- Tiêu đề: Playfair Display (đã có)
- Body: Inter (đã có)
- Giảm cỡ chữ tiêu đề, tăng line-height
- Sử dụng font weights khác nhau để tạo hierarchy

### 4.2 Spacing
- Tăng padding vertical giữa các section
- Giảm padding bên trong cards
- Sử dụng rem thay vì px

### 4.3 Visual Effects
- Gradient backgrounds cho hero
- Subtle animations (không quá mạnh)
- Border-radius đồng nhất

---

## 5. Thứ tự ưu tiên thực hiện

1. **Trang chủ** - Quan trọng nhất, ảnh hưởng đầu tiên
2. **Services page** - Cấu trúc alternating
3. **Solutions page** - Tab/accordion layout
4. **Pricing page** - Tinh gọn
5. **Các trang còn lại**
